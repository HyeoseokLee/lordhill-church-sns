# 코드 작성 규칙

## 주석

- 새로운 코드(함수, 클래스, 미들웨어, 라우트, 상수 블록 등)를 작성할 때 해당 코드 바로 위에 기능/역할을 설명하는 한 줄 주석을 달 것.
- 기존 코드를 수정할 때도 동작이 달라지면 주석을 추가하거나 업데이트할 것.
- 주석은 한국어로, 간결하게 작성 (예: `// 피드 목록 커서 기반 페이지네이션`).

## 코드 품질 검사

- 코드를 작성하거나 수정한 뒤 반드시 다음 두 명령을 실행하여 통과를 확인할 것:
  ```bash
  npm run lint      # ESLint
  npm run prettier  # Prettier
  ```
- 에러가 있으면 즉시 수정 후 재검사.
